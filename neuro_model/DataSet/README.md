# DataSet — Инструкция по работе

- **Папка:** `DataSet/`
- **Файлы:** `dataset.h`, `dataset.cpp`
- **Назначение:** Генерация синтетических данных для задач бинарной классификации. Предоставляет инструменты для создания датасетов с нормальным (гауссовым) распределением признаков.

## ПОДКЛЮЧЕНИЕ

```cpp
// В вашем коде:
#include "../DataSet/dataset.h"
```

## СТРУКТУРЫ ДАННЫХ

### Dataset — контейнер данных

```cpp
struct Dataset {
    std::vector<std::vector<double>> inputs;   // Признаки [n_samples × n_features]
    std::vector<std::vector<double>> targets;  // Целевые метки [n_samples × 1]
};
```

| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `inputs` | `vector<vector<double>>` | Матрица признаков. Каждый внутренний вектор — один пример. |
| `targets` | `vector<vector<double>>` | Матрица ответов. Каждый внутренний вектор содержит одно число (класс). |

## КЛАСС DatasetGenerator

Класс предоставляет статические методы для создания синтетических наборов данных, удобных для тестирования алгоритмов обучения.

### generate_gaussian() — Генерация гауссовых кластеров

Создаёт два кластера точек, распределённых по нормальному закону (распределение Гаусса). Кластеры разнесены в пространстве, что позволяет тестировать способность сети разделять классы.

```cpp
Dataset generate_gaussian(
    size_t n_samples = 100, 
    double cluster_std = 0.5,
    double separation = 2.0
);
```

**Параметры:**

| Параметр | По умолчанию | Описание |
| :--- | :--- | :--- |
| `n_samples` | `100` | Общее количество точек. Делится поровну между двумя классами (`n / 2`). |
| `cluster_std` | `0.5` | Стандартное отклонение (разброс) точек внутри кластера. |
| `separation` | `2.0` | Расстояние между центрами кластеров по оси X. |

**Возвращаемое значение:**
- `Dataset` с двумя признаками (координаты `x`, `y`) и бинарной меткой (`0.0` или `1.0`).

**Геометрия генерации:**
- **Класс 0 (метка 0.0):** Центр в точке `(+separation / 2.0, 0.0)`.
- **Класс 1 (метка 1.0):** Центр в точке `(-separation / 2.0, 0.0)`.

## ГЕНЕРАЦИЯ СЛУЧАЙНЫХ ЧИСЕЛ

### Функция `gaussian_random(double mean, double std_dev)`

```cpp
double gaussian_random(double mean, double std_dev) {
    double sum = 0.0;
    for (int i = 0; i < 12; i++) {
        sum += (double)rand() / RAND_MAX;
    }
    sum -= 6.0; 
    return mean + sum * std_dev;
}
```

- **Алгоритм:** Суммирует 12 равномерно распределённых случайных чисел `[0, 1]`.
- **Результат:** Получается величина, приближенная к нормальному распределению `N(mean, std_dev^2)`.
- **Инициализация:** Внутри `generate_gaussian` вызывается `srand(time(nullptr))` для уникальности данных при каждом запуске.
